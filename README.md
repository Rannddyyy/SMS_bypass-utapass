Introduction
---

將原先需手動開啟簡訊並點選 URL 進行二階段驗證的步驟轉為自動化，啟動此 App 後無需操作手機，只要接收到簡訊即立即自動化驗證(可在背景執行)。

此軟體的自動化流程為
1. 取得簡訊中的 URL (需先取得查看裝置簡訊的授權)
2. 透過 Webview 開啟網頁
3. 使用 javascript 點擊網頁中的按鈕完成驗證
